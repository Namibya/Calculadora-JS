# 🧮 Calculadora Simples (HTML, CSS e JavaScript)

Este projeto é uma calculadora básica desenvolvida utilizando **HTML**, **CSS** e **JavaScript puro**, com o objetivo de realizar operações matemáticas simples diretamente no navegador.
#Link para acesso
https://namibya.github.io/Calculadora-JS/
---

## 📌 Funcionalidades

- Operações básicas:
  - Adição (`+`)
  - Subtração (`-`)
  - Multiplicação (`×`)
  - Divisão (`÷`)
- Limpar o visor (C)
- Inserção de números e operadores via botões
- Atualização dinâmica do display

---

## 🏗️ Estrutura do Projeto

O projeto é dividido em três partes principais:

### 1. HTML (Estrutura)

Responsável por criar a interface da calculadora:

- Um **display** (visor) para mostrar os números e resultados
- Botões para números (0–9)
- Botões para operadores (+, -, ×, ÷)
- Botão de limpar (`C`)
- Botão de resultado (`=`)
